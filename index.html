SOBRE O JOOMLA

O que é Joomla

Joomla! é um sistema de gerenciamento de conteúdo CMS gratuito e de código aberto para publicação de conteúdo da web,Ele é construído em uma estrutura de aplicativo da web modelo-visualização-controlador que pode ser usada independentemente do CMS que permite criar aplicativos on-line.

SISTEMA DE GERENCIAMENTO CMS

Sobre o gerenciamento

A definição de CMS é um aplicativo baseado na web que fornece recursos para vários usuários com diferentes níveis de permissão para gerenciar todo ou uma seção de conteúdo, dados ou informações de um projeto de site ou aplicativo de intranet. Gerenciar conteúdo refere-se à criação, edição, arquivamento, publicação, colaboração, geração de relatórios e distribuição de conteúdo, dados e informações do site.

COMO DESENVOLVER

Desenvlvimento
